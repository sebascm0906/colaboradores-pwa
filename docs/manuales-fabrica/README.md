# Manuales rápidos para personal de fábrica

Esta carpeta tiene **manuales cortos** para el personal operativo de fábrica.

## Para qué sirven

- Te dicen **qué hacer paso a paso** dentro de la app.
- Están escritos en frases cortas.
- No explican cómo funciona el sistema por dentro.
- Si tienes duda, **avisa a tu supervisor**.

## Estos manuales NO son lo mismo que

- `docs/USER_MANUAL_BY_ROLE.md` — ese manual es para usuarios generales y supervisores. Tiene más detalle del que necesitas en planta.
- `docs/CODE_MANUAL.md` — ese es para programadores.

## Reglas básicas para todos

- Usa el celular asignado a tu turno.
- No prestes tu PIN a otra persona.
- Si la app marca un error, no insistas con el mismo dato.
- Si algo no aparece o no funciona, **avisa a tu supervisor**.
- Al terminar tu turno, sal de la app.
